## English

**Epusdt** (Easy Payment Usdt) is a self-hosted USDT payment middleware. Integrate USDT payments into any system via HTTP API — requires only MySQL/SQLite (and optionally Redis). No signup fees, no commissions. Funds go directly to your wallet.

**Features:** private deployment · multi-wallet polling · async queue · Telegram bot · cross-platform binary · plugin support

## 中文

**Epusdt**（Easy Payment Usdt）是一个由 Go 语言编写的私有化 USDT 支付中间件（TRC20 网络）。通过 HTTP API 将 USDT 收款集成到任意系统，仅依赖 MySQL/SQLite（可选 Redis），无需手续费，代币直达您的钱包。

**特点：** 私有部署 · 多钱包轮询 · 异步队列 · Telegram 机器人 · 跨平台二进制 · 插件支持
